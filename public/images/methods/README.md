# Images pour la Section "Mes Méthodes"

Ce dossier contient les images utilisées dans la section "Mes Méthodes" de la page `/approche`.

## Images requises

Chaque méthode nécessite une image de haute qualité (recommandé : 800x600px minimum).

### Liste des images nécessaires :

1. **micronutrition.jpg**
   - Suggestion : Suppléments nutritionnels, vitamines, superaliments
   - Thème : Science de la nutrition, analyses nutritionnelles

2. **functional-nutrition.jpg**
   - Suggestion : Aliments fonctionnels, système digestif, santé holistique
   - Thème : Approche globale de la nutrition

3. **glycemic-management.jpg**
   - Suggestion : Glucomètre, aliments à index glycémique bas, équilibre alimentaire
   - Thème : Gestion de la glycémie, prévention du diabète

4. **anti-inflammatory.jpg**
   - Suggestion : Aliments anti-inflammatoires (curcuma, poissons gras, légumes verts)
   - Thème : Nutrition anti-inflammatoire

5. **oligotherapy.jpg**
   - Suggestion : Plantes médicinales, oligoéléments, phytothérapie
   - Thème : Approche naturelle et complémentaire

## Spécifications techniques

- **Format** : JPG ou PNG
- **Dimensions recommandées** : 800x600px (ratio 4:3) ou supérieur
- **Taille de fichier** : < 500KB pour optimiser les performances
- **Style** : Photos professionnelles, lumineuses, cohérentes entre elles
- **Couleurs** : Tons naturels, verts doux en accord avec la palette NutriSensia

## Fallback

Si une image n'est pas disponible, le système utilisera automatiquement une image placeholder d'Unsplash.

## Suggestions de sources d'images

- Unsplash.com (images libres de droits)
- Pexels.com (images gratuites haute qualité)
- Photos personnelles si vous avez des images de votre cabinet ou de vos consultations
